# Wkhtml

This is an update of code from https://github.com/insphire/wkhtmltopdf-binary

It also bunles wkhtmltoimage

## Installation

Add this line to your application's Gemfile:

    gem 'wkhtmltox'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wkhtmltox
